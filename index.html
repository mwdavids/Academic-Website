Data and Code

I work primarily in R, and have made public some code that might be helpful for others:
  1. Multidimensional Measures of Social Structure. These functions will calculate fractionalization, bipolarization, cross-cuttingness measures from survey or census data. [Under development][Paper by Joel S. Selway]
  2. Proportional Representation Seat Calculator. This code calculates the number of seats different parties should be awarded given the number of votes they have received.
